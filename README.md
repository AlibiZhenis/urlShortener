# URL Shortener Microservice

This is Alibi Zhenis's personal project.
Tech stack used: Node.js, Express.js, MongoDB, Mongoose

# Setup
1. Set an environment variable MONGO_URI to your database's connection string.
2. Launch command:
``node index.js``
